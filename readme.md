### ACL  
  
ACL can get true, false, enable or defined name  
  
#### ex  

    {  
      ...  
      "acl":true,         // disable element  
      "acl":false,        // enable element for every one 
      "acl":"enable",     // enable if user is signein 
      "acl":"beta_test",  // enable if user have permision 
      ...  
    }  
  
#### prototype
  
    { 
      "enable": true,

      "debug": true,  
      "setting": true,
