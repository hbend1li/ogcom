### ACL  
  
ACL can get true, false, signin or defined name  
  
#### ex  

    {  
      ...  
      "acl":true,  
      "acl":false,  
      "acl":"signin",  
      "acl":"beta_test",  
      ...  
    }  
  
#### prototype
  
    {  
      "admin": true,  
      "beta": true,  
      "debug": true,  
      "setting": true,  
      "user_list": true,  
      "product": true,  
      "purchase": true,  
      "sale": true  
    }
